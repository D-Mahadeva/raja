Platform Clone Integration

This is a simplified clone of Zepto for demonstrating checkout integration with PriceWise.

How it works

  1. PriceWise sends cart data to this Zepto clone when a user chooses to checkout
  2. The user completes payment process on this platform
  3. Order status is sent back to PriceWise in real-time
  4. The user can track their order from either platform

Test this integration

Add items to your cart in PriceWise, then select Zepto as your checkout platform.
